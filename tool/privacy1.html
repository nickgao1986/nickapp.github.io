一、引言

夜拍补光助手（以下简称「本 App」）是一款以拍摄补光为主的工具类应用：支持人像柔光、食物暖光、冷白清晰、证件照均匀光等场景预设，提供色温与亮度调节、构图网格与水平线辅助；并在工具页提供手电筒、频闪与 SOS 等应急能力（具体能力以您设备与系统版本为准）。

本隐私政策说明本 App 如何对待您的信息。本 App 不运营用户账号体系，不会将您的补光参数、收藏方案、工具偏好等个人使用数据上传至开发者服务器；核心数据保存在您的设备本地（及本说明所述的 Apple 系统机制范围内）。请您在使用前仔细阅读本政策。

二、我们不做的数据收集

本 App 为以本地功能为主的工具类应用，开发者不通过本 App 主动收集以下信息用于用户画像、广告追踪或云端存档：

  1. 身份与联系方式：不要求注册、登录，不收集姓名、手机号、邮箱等；
  2. 补光参数与收藏方案：色温、亮度、网格等设置及您保存的方案仅存本地，用于恢复您的使用习惯；
  3. 应急工具偏好：手电筒亮度、频闪与 SOS 相关选项仅存本地；
  4. 精确地理位置、通讯录：本 App 不因上述核心功能而采集此类信息（若系统版本或未来功能有变化，将以更新后的本政策为准）。

三、设备权限与本地数据

为实现功能，本 App 可能在您授权后使用下列系统能力；相关说明以系统弹窗与 Apple 官方文档为准：

  • 手电筒 / 闪光灯：通过系统 API 控制设备补光灯用于照明；不会因手电筒功能而采集您的照片或视频内容。
  • 麦克风：若您使用「录制语音」作为自定义闹钟铃声，音频保存在本机应用沙盒，用于播放提醒；开发者不接收录音文件副本。
  • 相册（照片库）：若您使用壁纸相关能力选择图片或保存导出作品，由系统在授权范围内访问相册；开发者不远程读取您的相册。
  • 屏幕使用时间 / 专注模式（如适用）：若您启用相关能力，系统可能要求额外授权；与本 App 相关的配置与选择结果按 Apple 机制存储，用于在设备上执行您选择的限制策略。

以下数据会保存在您的设备上（例如通过 UserDefaults、本地文件或系统提供的存储方式），不会由开发者远程读取：

  • 闹钟与提醒：闹钟列表、重复规则（含法定工作日与内置节假日参考数据）、铃声与稍后提醒等；
  • 补光与构图偏好：色温、亮度、场景预设、网格与水平线开关、收藏方案等；
  • 应急工具偏好：手电筒、频闪与 SOS 相关设置；
  • 应用偏好：语言、浅色/深色显示、主题色、引导是否完成等；
  • 为改善体验的本地计数：例如启动次数、上次请求系统评分提示的时间等（仅用于控制提示频率，不上传）。

1. 系统通知

闹钟响铃、稍后提醒、专注结束等依赖 Apple UserNotifications 框架在本地调度通知。您可在系统「设置 → 通知」中管理权限。通知内容由本机在本地生成，开发者不接收通知内容副本。

2. 桌面小组件与锁屏小组件（Widget）

若您添加本 App 提供的小组件，小组件通过 Apple 提供的 App Group 等机制读取已在您设备上保存的摘要信息（例如下次闹钟或快捷打开手电筒所需的最小数据），用于在桌面或锁屏展示。该共享容器位于您的设备内，用于主 App 与扩展之间同步展示所需数据，开发者无法远程访问。

3. 实时活动（Live Activities，如适用）

若系统支持且您开启相关功能，锁屏或灵动岛可能展示闹钟或专注进行中的状态摘要；数据来源于本机调度状态，不上传至开发者服务器。

4. 存储空间与缓存

您可在设置内使用「存储空间」相关能力查看占用，并对缓存、临时文件等进行清理；操作均在本地完成。

四、跳转与系统界面

  • App Store：「喜欢我们就给个好评」可能打开 App Store 评价页；「更多应用」可能打开 Apple 上的其他应用链接。上述跳转由系统浏览器或 App Store 处理，开发者不因此获知您的 Apple ID 或购买明细。
  • 分享：使用系统分享面板时，由您选择分享对象与渠道，开发者不截获分享内容。

五、网络与第三方 SDK

本 App 不集成用于广告追踪、用户行为画像的第三方数据分析 SDK。除您主动点击外链（如上述 App Store 页面）外，不因核心功能而向开发者服务器上传您的个人数据。

六、未成年人

本 App 面向普通用户。若您为未成年人，请在监护人指导下阅读本政策并使用。

七、您的控制与删除
