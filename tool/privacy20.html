音频合并录音工具

导入与麦克风录制多段音频合并导出隐私政策

生效日期：2026年2月2日

一、概述

音频合并录音工具（以下简称本应用）是一款运行在 Apple macOS 上的桌面软件，用于在本地导入音频文件或使用麦克风录制音频片段，按列表顺序进行音量调节与可选交叉淡入等处理后，合并导出为新的音频文件（例如 AAC m4a）。本政策说明本应用在您设备上处理数据的方式。若您继续使用本应用，即表示您已阅读并理解本政策。

二、我们收集的信息

本应用不运营用户账户体系，不要求您登录，不会因您使用核心合并与导出功能而向开发者服务器上传您的录音内容或导入的音频文件用于识别个人身份或用于广告画像。

在您使用本应用的过程中，可能涉及以下与设备本地相关的信息类型，均发生在您的 Mac 上：

  • 您主动导入或录制的音频：当您通过系统文件选择器从本机选取音频文件时，内容由 macOS 授权可读的路径决定；当您使用麦克风录制时，音频信号在本机采集并写入应用沙盒或您通过导出功能指定的目标位置，录制目的仅限于您在本应用内的合并与导出操作。
  • 应用运行所需的本地数据：包括合并列表中的片段顺序、音量与交叉淡入等与本机项目相关的设置，以及为保障功能正常运行而生成的临时或缓存文件。上述内容默认保存在本机应用容器目录内，由您通过系统与应用界面掌控。

三、数据存储位置

导入副本、录音片段、合并过程中的中间结果以及您未另行指定路径前的导出结果，通常保存在 macOS 为本应用分配的应用沙盒目录下（例如「应用程序支持」等路径中的应用专属子目录）。当您使用系统的「存储到」或保存面板选择导出位置时，文件将写入您明确授权的目标文件夹。上述数据不会由本应用自动同步到我们的云端服务器。请您自行通过 Time Machine 或其他备份方式保护重要音频。

四、网络访问

本应用的导入、录音、预览、合并与导出等核心流程可在无互联网连接的环境下完成。若未来版本因检查更新或可选崩溃诊断等引入联网能力，我们将在本政策中补充说明具体用途、数据类型与关闭方式。

五、第三方服务与分析工具

当前版本不包含广告模块，不集成第三方统计或社交分享 SDK。若后续版本发生变化，我们将更新本政策并在合理位置提示您。

六、权限说明

本应用可能请求或使用以下与 macOS 系统相关的权限，均在您触发对应功能时由系统提示并由您授权：

  • 麦克风：仅在您开始录音时用于采集音频，停止录音或退出相关界面后不再持续监听。
  • 用户选择的文件：通过标准打开或保存对话框访问您指定的音频文件或导出路径；除您发起操作所涉及的文件外，本应用不会擅自批量扫描磁盘其他位置。

七、未成年人

本应用面向需要在 Mac 上整理与合并音频的成年用户及经监护人同意的未成年人。未成年人请在监护人指导下使用麦克风录音功能并注意勿录制他人私密谈话。

八、您的控制与删除

您可随时在本应用内移除列表中的片段，也可在访达中删除应用沙盒目录或您自行选择的导出文件夹中的音频文件。卸载本应用后，沙盒内数据将按 macOS 规则处理，具体以系统版本为准；若您曾将文件导出到文稿或外置磁盘，需自行管理该副本。

十一、常见问题（FAQ）

1）我的录音或导入的音频会上传到你们服务器吗？

不会用于上述目的。当前版本的核心功能在本地完成，我们不会以识别您个人身份为目的将您的录音或导入音频自动批量上传到开发者服务器。

2）麦克风什么时候会被使用？

仅在您在本应用内主动开始录音时使用，用于将音频保存到本机以便后续合并或导出。

3）如何删除本地音频数据？

您可以在应用内删除列表条目，或在访达中删除应用数据目录及您自行保存的导出文件。卸载应用后请根据 macOS 说明管理残留数据。

4）合并失败或导出异常怎么办？

请发送邮件到支持邮箱，并附上 macOS 版本、应用版本、问题描述与复现步骤，我们会协助排查。

摘要：音频合并录音工具在您的 Mac 上完成音频导入、录音、列表合并与导出。数据默认保存在本机，本应用不以识别您个人身份为目的向开发者服务器上传您的音频内容。麦克风与文件访问遵循系统授权。
