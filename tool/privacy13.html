Mac清理助手

照片清理与图片工具隐私政策

生效日期：2026年4月18日

一、概述

Mac清理助手（以下简称本应用）是一款运行在 Apple macOS 上的桌面软件，提供照片清理、相册管理与图片工具等能力。本政策说明本应用在您设备上处理数据的方式。若您继续使用本应用，即表示您已阅读并理解本政策。

二、我们收集的信息

本应用不运营用户账户体系，不要求您登录，不向开发者服务器上传您的项目内容或截图文件用于识别个人身份或用于广告画像。

在您使用本应用的过程中，可能涉及以下与设备本地相关的信息类型，均发生在您的 Mac 上：

  • 您主动创建的内容：包括项目名称、每张截图的标题与说明文字、主题与版式相关设置、导出尺寸与机型展示选项、截图与可选自定义外框素材对应的文件等。上述内容默认保存在本机应用支持目录中，由您完全掌控。
  • 您从磁盘选择的文件：当您通过系统文件选择器导入图片或选择导出文件夹时，由 macOS 与您的操作授权决定可读写的路径。本应用仅在您发起相关操作时访问您选定的文件或目录。

三、数据存储位置

您的配置与处理结果通常以文件形式保存在本机应用支持目录（例如 AppPic 或 MacCleaner 相关目录）或您导出时选择的目标路径中。上述数据不会由本应用自动同步到我们的云端服务器。请您自行通过 Time Machine 或其他备份方式保护重要数据。

四、网络访问

本应用的核心编辑、预览与导出流程可在无互联网连接的环境下使用。若未来版本因检查更新或崩溃报告等引入可选联网能力，我们将在本政策中补充说明具体用途、数据类型与关闭方式。

五、第三方服务与分析工具

当前版本不包含广告模块，不集成第三方统计或社交分享 SDK。若后续版本发生变化，我们将更新本政策并在合理位置提示您。

六、权限说明

本应用通过 macOS 标准文件对话框访问您选择的图片与导出目录。除您明确发起的导入与导出操作外，本应用不会扫描您磁盘上的其他无关目录。

七、未成年人

本应用主要面向具备应用上架与视觉设计需求的成年用户。若未成年人在监护人同意下使用，监护人应指导其注意本地文件与导出内容的管理。

八、您的控制与删除

您可随时在本应用内删除项目或截图条目，也可在访达中删除应用支持目录下的相关文件。卸载本应用后，除非您另行备份，上述本地数据将随卸载由系统处理，具体行为以 macOS 版本规则为准。

九、政策更新

我们可能因功能调整或法律法规要求更新本政策。更新后的版本将替换本页面所示文本，并调整生效日期。若变更重大，我们将在应用内或本页面以合理方式提示。

十一、常见问题（FAQ）

1）我导入的截图会上传到你们服务器吗？

不会。当前版本的截图导入、编辑与导出流程主要在本地完成，我们不会以识别个人身份为目的将您的项目截图自动上传到开发者服务器。

2）如何删除本地数据？

您可以在应用内删除项目与图片，或在 macOS 访达中删除本应用的本地数据目录。卸载应用后，系统会按 macOS 规则处理残留数据。

3）如果遇到无法导出或显示异常怎么办？

请发送邮件到支持邮箱，并附上 macOS 版本、应用版本、问题截图和复现步骤，我们会协助排查。

摘要：Mac清理助手在本地帮助您进行照片清理与图片处理。您的图片与处理结果默认保存在本机，本应用不以识别您个人身份为目的向开发者服务器批量上传上述内容。导入与导出依赖您在本机授权的文件路径。
