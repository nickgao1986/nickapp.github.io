一、引言

美颜相机（以下简称"本App"）是一款屏幕补光与拍照工具类应用，主要提供将手机屏幕用作补光灯、多种预设光效与自定义调光、拍照、Live Photo、录像等功能；部分高级功能（如更多场景、光效模式、取景框形状等）可通过可选的应用内购买解锁。

本隐私政策旨在向您说明本App如何收集、使用、存储与保护您的信息，以及应用内购买相关的数据处理方式。本App以隐私为先，除应用内购买所需与苹果交互的购买状态外，不收集、不存储、不上传、不共享您的个人身份信息或拍摄内容，相关处理均在您设备本地完成。请您在使用本App前仔细阅读并理解本隐私政策，使用即视为接受本政策。

二、信息收集与使用

本App为以本地运行为主的工具类应用，仅在以下有限情形下与外部发生数据交互，其余均不收集用户数据。

2.1 应用内购买相关

当您使用本App提供的可选应用内购买（内购）功能时：

  • 购买与支付：所有付款均由苹果公司（Apple）通过 App Store 与 StoreKit 处理。本App不收集、不存储、不接触您的支付信息（如银行卡、支付宝、Apple Pay 等），也不会收到您的完整账号或支付凭证。
  • 购买状态（权益）：本App仅通过苹果提供的 StoreKit 接口获知您是否已购买「高级会员」等内购产品，用于在您设备上解锁相应功能。该购买状态由苹果与您设备本地管理，我们不会将该状态与您的身份信息关联或上传至自有服务器。
  • 恢复购买：您可通过本App内的「恢复购买」功能，在更换设备或重装App后恢复已购权益。该过程由苹果验证您当前 Apple ID 的购买记录，本App仅根据苹果返回的结果更新本地解锁状态，不额外收集或上传数据。

内购产品类型为非消耗型（一次性购买，永久有效）。您无需购买即可使用本App的核心补光、拍照与录像功能；内购仅用于解锁更多场景、光效、取景框等高级功能。

内购产品与解锁内容说明：本App提供的应用内购买产品为「高级会员」（或产品列表中展示的等效名称），类型为非消耗型，一次性付款后永久有效。购买后解锁的内容包括但不限于：更多补光场景、更多预设颜色与光效模式、更多取景框形状、超级亮模式、高级色阶与饱和度调节、更多配色方案保存数量及颜色历史与收藏等。具体以App内展示为准。付款与订阅管理均由苹果 App Store 完成，您可在系统「设置 → Apple ID → 订阅」中查看或管理；本App不存储、不接触您的支付方式或账单信息。

2.2 我们不收集的信息

除上述内购相关的最小必要信息外，本App不收集以下任何信息：

  1. 个人身份信息：如姓名、手机号、邮箱、头像、昵称等；
  2. 设备信息：如设备型号、操作系统版本、设备ID、IP地址、地理位置、网络状态等；
  3. 使用行为信息：如使用时长、操作记录、功能使用偏好、点击记录等；
  4. 拍摄内容：您使用本App拍摄的照片、Live Photo、视频仅保存在您的设备相册或本地预览，我们不会收集或上传；
  5. 人脸与图像数据：补光预览、肤色分析等均在设备本地实时处理，不存储、不上传任何图像或人脸数据；
  6. 本地设置：语言、主题、取景框形状、补光颜色等偏好仅存储在您的设备本地，卸载App后将自动清除。

三、权限说明

本App仅申请实现功能所必需的权限，所有与拍摄、保存相关的数据均仅在您设备本地处理：

  • 相机权限：用于拍照、拍 Live Photo、录像及补光画面预览；所有画面仅在您设备上显示，不上传；
  • 相册写入权限：仅用于将您拍摄的照片与视频保存到系统相册；本App不读取您相册中的其他内容，数据仅存储在您的设备；
  • 麦克风权限（如适用）：仅用于录像时录制声音，音视频均保存在本地，不上传。
